= K8S Installer
:github: https://github.com/ten1010-io/k8s-installer

== 개요

K8S Installer는 오프라인 환경에서 쿠버네티스 클러스터를 구성하는 도구입니다

== 지원하는 운영체제

* Ubuntu 22.04
** 22.04.0 ~ 22.04.5 지원
* Redhat 8
** 지원하지 않음
* Centos 8
** 지원하지 않음
* CentOS 8 Stream
** 지원하지 않음
