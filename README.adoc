= K8S Installer
:github: https://github.com/ten1010-io/k8s-installer

== 개요

K8S Installer는 오프라인 환경에서 쿠버네티스 클러스터를 구성하는 도구입니다

== 지원하는 운영체제

* Ubuntu 22.04
** minor 버전 0부터 3까지(22.04.0 ~ 22.04.3)
* Redhat 8
** minor 버전 0부터 7까지(8.0 ~ 8.7)
* Centos 8
** minor 버전 0부터 7까지(8.0 ~ 8.7)
** CentOS 8 Stream은 지원하지 않습니다
